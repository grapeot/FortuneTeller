# 面相学知识库（Prompt 参考文档）

本文档整理自传统面相学资料，用于构建 AI 算命师的系统 prompt。
目标：让 AI 能根据真实人脸照片给出 specific 的面相论断，而非泛泛的吉祥话。

---

## 一、五官定义（面相术语）

| 官名 | 部位 | 术语 | 主管 |
|------|------|------|------|
| 采听官 | 耳 | 轮、廓、垂珠、风门 | 长寿、冒险精神、童年运 |
| 保寿官 | 眉 | 凌云、紫气、繁霞、彩霞、眉棱骨 | 健康、地位、兄弟朋友运 |
| 监察官 | 眼 | 龙宫、凤池、鱼尾、奸门、卧蚕、泪堂 | 意志力、心性、智慧、领导力 |
| 审辨官 | 鼻 | 山根、年上、寿上、准头、兰台、廷尉 | 财富、健康、自我力量 |
| 出纳官 | 口 | 海角、人中、唇、承浆 | 幸福、食禄、贵人运、晚年 |

## 二、三停（面部三分区）

| 分区 | 范围 | 年龄段 | 主管 |
|------|------|--------|------|
| 上停 | 发际→眉毛 | 15-30岁 | 智慧、父母运、早年学业事业 |
| 中停 | 眉毛→鼻准头 | 31-50岁 | 事业、财富、中年运 |
| 下停 | 鼻下→下巴 | 51岁以后 | 晚年福气、子女运、健康 |

**判断标准：**
- 上停方正广阔 → 早年顺遂，社会地位高
- 中停丰隆端峻 → 寿命长久，事业有成
- 下停圆实丰厚 → 晚年安乐，福寿齐全
- 哪个区域最大/最突出 → 该区域对应的特质最强

## 四、各部位详细相法

### 4.1 额头（天庭）
- **饱满高广** → 智慧高、早年运佳、有领导潜质
- **窄小低陷** → 早年运势受阻
- **方正** → 决策能力强、有格局
- **有清晰纹路** → 思虑深、有经历
- 额头占面部比例大 → "思维型"人格，喜欢分析利弊

### 4.2 眉毛（保寿官）
- **浓密、形清晰、眉间距两指宽** → 坚持己见、决策果断、适合管理
- **如新月、细长过目** → 待人宽和、聪明
- **稀疏、眉色淡** → 性格温和、易受影响、适合执行
- **眉间距窄/连眉** → 执着、对事放不下心
- **眉头清晰眉尾散** → 中年才开始发力
- **眉高** → 自尊心强
- **左眉强于右眉** → 私下更有决心；反之公开场合更强势

### 4.3 眼睛（监察官）
- **大而有神** → 富同情心、情感细腻、人际关系好
- **黑白分明** → 心思纯正、有才华智慧
- **细长** → 理性冷静、判断公正
- **含藏不露** → 内敛有深度
- **单眼皮** → 理性；**双眼皮** → 感性
- **卧蚕饱满** → 子女运好、有异性缘
- **眼尾上翘** → 有魄力；**下垂** → 温顺

### 4.4 鼻子（审辨官/财帛宫）
- **鼻梁高挺** → 自尊心强、官运通
- **准头（鼻头）有肉、圆润** → 正财运好、会存钱
- **鼻翼饱满** → 能聚财
- **鼻孔仰露** → 财来财去、花钱大方
- **山根高** → 自信、意志力强
- **山根低** → 需要更多后天努力
- **鼻梁有节** → 多疑
- 鼻子是面部最高点，代表40-50岁黄金事业期

### 4.5 嘴巴（出纳官）
- **大而有收拾** → 有福气、有口才
- **唇红润厚实** → 重感情、食禄丰厚
- **嘴角上翘（弓形）** → 乐观、人缘好
- **嘴角下垂** → 不满足
- **上唇薄下唇厚** → 注重物质享受
- **上下唇均匀** → 性格平衡
- **人中深直** → 长寿、生育力强

### 4.6 下巴/下颌
- **饱满有光泽** → 晚年福运好、下属缘好、管理能力强
- **方阔** → 意志坚定、领导力强
- **尖薄** → 晚年需多积蓄
- **双下巴** → 长寿之相
- **圆润** → 适应力强

### 4.7 颧骨
- **高而有肉** → 有权力欲、管理能力强
- **颧骨突出无肉** → 性格刚烈但孤
- **左颧强** → 公共场合更有掌控力
- **右颧强** → 家庭中更有话语权

### 4.8 法令纹
- **深长** → 位高权重、长寿
- **八字形宽广** → 行动力强、受人信赖
- **前端变窄** → 需多努力
- **30岁前出现** → 早熟、独立
- **40岁后出现** → 正常

### 4.9 耳朵
- **轮廓分明** → 学习能力强
- **耳垂厚大** → 福气好、金钱运好
- **贴面** → 性格内敛稳重
- **招风** → 性格外向独立
- **耳高与眉齐** → 聪明

## 五、脸型与性格

| 脸型 | 特征 | 性格 |
|------|------|------|
| 方形 | 宽度≈长度 | 领导者、行动导向、坚持己见 |
| 矩形/椭圆 | 比方形更长窄 | 管理者、贵族气质、政商精英 |
| 圆形 | 宽而短 | 社交达人、水形人、适应力强 |
| 长形 | 窄而长 | 学者型、思考深入 |
| 三角形（倒） | 额宽下窄 | 聪明但固执 |
| 菱形 | 颧骨最宽 | 有权力欲、独立 |

## 六、面相口诀速查

- **看慧在额**：天庭饱满 → 智慧
- **看贵在眼**：锐利有神 → 贵人运
- **看富在鼻**：丰大挺拔 → 财富
- **看寿在颌**：方阔圆润 → 长寿
- **看名在眉**：清秀名扬 → 地位
- **看福在耳**：厚耳招福 → 福气
- **看禄在嘴**：丰润红唇 → 食禄

**相术总口诀：**
> 邪正看眼鼻，真假看嘴唇；功名看气概，富贵看精神；主意看指爪，风波看脚筋；若要看条理，全在语言中。

## 七、在微软场景下的映射

| 面相特征 | 微软职场映射 |
|----------|-------------|
| 额头饱满高广 | 架构思维强，适合 Principal/Architect |
| 眉毛浓密清晰 | 决策果断，Design Review 一针见血 |
| 眼神锐利有深度 | Code Review 细致，Bug一眼看穿 |
| 鼻梁高挺准头有肉 | 财帛宫旺，年终 bonus 丰厚 |
| 嘴角上翘有收拾 | 表达能力强，适合 PM 或 Tech Lead |
| 下巴方阔饱满 | 带团队能力强，1:1 有说服力 |
| 颧骨高有肉 | 管理欲望强，适合走 Manager 路线 |
| 法令纹深长 | 在组织中有影响力，Story Points 说了算 |
| 耳垂厚实 | 运气好，On-call 不出 P0 |
